# CAST·SPLINT 생산공정일지 관리 시스템

수기 공정일지(F-PD-003A Rev.12)와 엑셀 실적 정리를 대체하는 로컬 웹 시스템입니다.

## 공정 구분 (CAST / SPLINT)

대시보드·생산계획표·생산실적·분석 화면 상단의 **[CAST | SPLINT] 토글**로 공정을 전환합니다.
두 공정은 데이터·입력 양식·계산식이 분리되어 있습니다.

- **CAST**: EA 단위. 로스율 = 불량 ÷ 총생산량(loss포함)
- **SPLINT**: roll 단위(SP내수/SP해외/PR). 로스율 = 불량(roll) ÷ 이론총수량(roll)
  - PR(roll) = PR(m) ÷ 1롤길이(기본 4.55m) · 이론총수량 = 기재투입(m) ÷ 1롤길이
  - 총로스(roll) = 총로스(g) ÷ 제품무게 · 생산불량 = 총로스 − 공정불량
  - **제품무게 측정 기록**: 생산자가 측정한 무게(시간/값/비고)를 여러 건 입력 → 평균·최소·최대 자동 계산,
    [평균을 제품무게 1개(g)에 적용] 버튼으로 로스 환산 기준 무게에 반영

## 실행 방법

```
cd test1
node server.js
```

브라우저에서 http://localhost:3000 접속

- 별도 설치(npm install) 없이 Node.js만 있으면 실행됩니다.
- 데이터는 `data/db.json` 파일에 저장됩니다. (백업 시 이 파일만 복사)

## 작업지시서

생산계획표의 [🖨 지시서] 버튼 → 계획 정보 + 제품표준서(제품명·칼라·업체로 자동 매칭)를 결합한 작업지시서가 생성됩니다.
생산계획 / 자재 기준(기재·수지·토너·파우치) / 포장 기준(라벨·In Box·Out Box, 사진 포함) / 결재란 구성이며 [인쇄] 버튼으로 A4 출력할 수 있습니다.
사진은 업로드 시 1280px로 자동 축소되어 `public/uploads/`에 저장됩니다.

## 업무 흐름 (실적은 직접 입력하지 않습니다)

1. **생산계획표**에 계획 등록
2. 생산일에 **일일 공정일지** 작성 → [생산계획 불러오기]로 라인 자동 채움 → 저장
3. 저장하는 순간 **생산실적이 자동 생성**되고(자동계산 + 제품표준서에서 기재·수지 자동 보완), 계획표에 달성률 표시
4. 실적을 고치려면 실적 행을 클릭 → **원본 공정일지가 열림** → 일지에서 수정하면 실적이 다시 반영
5. 엑셀에서 이관된 과거 실적(공정일지 없음)만 예외적으로 직접 보완 가능

생산실적 탭은 조회·필터·CSV 내보내기 전용입니다.

## SPLINT 공정 (ROLL 제품)

SPLINT는 ROLL 제품이며, 길이·무게 기준 미달품은 폐기하지 않고 **PRECUT(반제품, 짧은 길이)** 생산으로 전환됩니다.
생산중 기록 탭(태블릿 주 사용 화면)은 **제품별 생산실적**을 중심으로 구성합니다.
- **제품별 생산실적**: [제품 추가]로 제품마다 제품명·생산량·작업 로스·비고를 입력. 제품 1건 = 생산실적 1건(레코드)으로 저장되어 제품별 불량률 분석 가능. 제품명에 PRECUT/프리컷이 들어가면 자동으로 PRECUT로 분류.
- **자동 계산은 총 로스량(= 모든 제품 작업 로스 합계)** 만 강조. 총 생산량/총 ROLL/총 PRECUT은 별도로 합산 표시하지 않고 제품별 목록을 우선합니다.
- 제품무게 측정(시간칸 HH:mm 넓게), 중피·상지·하지 교체 이력([지금] 버튼·수동 수정·여러 건·비고).
정기점검(②탭)은 휴게 후 재가동 전 확인용으로 10:30 / 15:30 두 시간대, 온도·습도를 각각 분리 입력합니다.
생산종료 탭에는 제품별 생산실적 목록과 총 로스량(+평균무게·기준이탈)이 자동 반영됩니다.

## CAST 공정 (작업흐름형)

CAST도 SPLINT와 동일한 탭 구조·UX입니다.
- **생산준비/정기점검**: 10:30 / 15:30 두 시간대, 온도·습도 분리 입력 (휴게 후 재가동 전 확인).
- **생산중 기록**(태블릿 주 사용): 제품별 생산실적([제품 추가]로 제품명·생산량·작업 로스·비고), 기재 교체 이력([지금] 버튼·수동수정·여러 건·비고), 기포 테스트 기록(시간·결과·비고), 수지 정보(수지명·컬러·합성일·배합일·촉매·번호·무게·비고), 특이사항. **총 로스량 = 모든 제품 작업 로스 합계** 자동 계산.
- **작업 로스는 생산실적 데이터**로 생산중 기록 탭에서 관리(품질확인 탭에는 품질 항목만).
- **생산종료**: 제품별 생산실적·총 로스량·기재 교체 이력·기포 테스트 결과·수지 정보가 자동 반영.

## 자동계산 (기존 엑셀 수식과 동일)

- 총생산량 = 생산수량(정품) + 잔량
- 총로스 = 공정불량 + 생산불량
- 총생산량(loss포함) = 총생산량 + 총로스
- 공정/생산/총 로스율(%) = 각 불량 ÷ 총생산량(loss포함) × 100
- 투입기재(m) = 총생산량(loss포함) × 길이
- 총 사용량(롤) = 투입기재 ÷ 기재길이
- 수지 총투입량(kg) = 투입기재 × 투입량1ea(g) ÷ 1000
- 파우치 총수량 = 생산수량(정품) + 파우치추가
- 불량 세부유형 입력 시 공정불량(이음매+중간매듭+기재불량), 생산불량(셋팅+말림+줄감+길이+오염+기타) 자동합산
